# Health Analytics Project

Full-stack application for health data analytics.

## Structure
- `backend/` - Spring Boot backend API
- `frontend/` - Angular frontend application

## Setup
1. Backend Setup
   - Navigate to \`backend\`
   - Run \`mvn spring-boot:run\`

2. Frontend Setup
   - Navigate to \`frontend\`
   - Run \`npm install\`
   - Run \`npm start\`
